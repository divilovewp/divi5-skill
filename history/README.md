# Version history (archive)

These are earlier, superseded versions of the Divi 5 Skill, kept for transparency so you
can see how the skill evolved. **They are not maintained — use the current skill in the
repo root.** See [`../CHANGELOG.md`](../CHANGELOG.md) for the change history.

Early releases were originally labelled "V1" and "V2"; the project later adopted semantic
versioning, so those are recorded here as **0.1.0** and **0.2.0**.

| File | Version | Divi Builder | Notes |
|------|---------|--------------|-------|
| [v0.1.0.md](v0.1.0.md) | 0.1.0 (was "V1") | 5.0.1 | First version. Single-file skill, JSON import/export workflow. |
| [v0.2.0.md](v0.2.0.md) | 0.2.0 (was "V2") | 5.0.1 | Reorganized; added flexbox/row deep-dive, card patterns, Python generator. |
| [v0.2.1-beta.md](v0.2.1-beta.md) | 0.2.1 (Beta) | 5.0.1 | Adds nested-rows, row/column/group card patterns, contact-form submit button. |

The lineage continued with **0.3.0** (Divi 5.6.2) — the first multi-file split — and the
current **0.5.x** line (Divi 5.7.x). See the repo root and `CHANGELOG.md`.

> Note: `v0.1.0.md` was reconstructed from the original document (character-encoding
> repaired) and, like all files here, has had site-specific brand references removed.
